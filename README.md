# Web版刷题练习系统

一个功能完整的Web刷题练习应用，支持离线运行、Excel题库导入、多种答题模式和智能错题集管理。

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问: http://localhost:5173/

### 构建生产版本

```bash
npm run build
```

## Excel题库格式

创建Excel文件时，请按以下格式填写（从第一行开始）：

| 题目类型 | 题目内容 | 选项A | 选项B | 选项C | 选项D | 正确答案 | 解析 |
|---------|---------|-------|-------|-------|-------|---------|------|
| 选择题 | JavaScript中哪个方法用于向数组末尾添加元素？ | push() | pop() | shift() | unshift() | A | push()方法向数组末尾添加一个或多个元素 |
| 选择题 | CSS中哪个属性用于设置元素的背景颜色？ | color | background-color | bgcolor | background | B | background-color是CSS标准属性 |
| 判断题 | HTML是一种编程语言 | | | | | 错 | HTML是标记语言，不是编程语言 |

**格式说明**:
- 第1列: 题目类型（必填，值为"选择题"或"判断题"）
- 第2列: 题目内容（必填）
- 第3-6列: 选项A/B/C/D（选择题必填，判断题可留空）
- 第7列: 正确答案（必填，选择题填A/B/C/D，判断题填"对"或"错"）
- 第8列: 解析（可选）

## 主要功能

✅ Excel文件导入（支持拖拽上传）  
✅ 题库管理（查看、删除）  
✅ 两种答题模式（顺序/乱序）  
✅ 选择题和判断题支持  
✅ 实时答题反馈  
✅ 智能错题集管理  
✅ 数据本地持久化  
✅ 完全离线运行  
✅ 现代化UI设计  
✅ 响应式布局  

## 技术栈

- React 18.3 + Vite 5.4
- Tailwind CSS 3.4
- React Router DOM 6.26
- SheetJS (xlsx) 0.18
- IndexedDB (idb) 8.0
- Lucide React (图标)

## 浏览器支持

- Chrome/Edge (推荐)
- Firefox
- Safari

需要支持 IndexedDB 和 ES6+ 特性。

## 数据存储

所有数据存储在浏览器的 IndexedDB 中，包括：
- 题库数据
- 答题记录
- 错题集

⚠️ 清除浏览器数据会导致题库丢失，请定期备份原始Excel文件。

## 使用流程

1. 导入题库：点击"选择文件"或拖拽Excel文件到上传区
2. 选择题库：在题库列表中点击"开始练习"
3. 选择模式：选择顺序模式或乱序模式
4. 开始答题：点击选项提交答案，查看反馈
5. 查看统计：完成所有题目后查看答题统计
6. 复习错题：在错题集页面进行针对性练习
